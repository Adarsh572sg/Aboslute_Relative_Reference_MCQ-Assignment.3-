# Absolute and Relative Reference MCQ Assignment

This repository contains the Multiple Choice Questions (MCQ) assignment focused on Absolute and Relative References. The assignment is designed to test your understanding of how absolute and relative references work in various contexts, likely within the scope of data analysis or spreadsheets.

## Introduction

Understanding the difference between absolute and relative references is crucial for anyone working with data, especially in fields like data analysis, programming, and spreadsheet management. This assignment aims to reinforce these concepts through practical MCQs.

## Assignment Structure

The assignment is structured using Jupyter Notebooks. Each notebook contains a series of questions that you need to answer. The questions are designed to test your knowledge of absolute and relative referencing techniques.
